SSC Ranker Up — Government Exam Prep App

A comprehensive Android application designed to help aspirants prepare for SSC (Staff Selection Commission) examinations including CGL, CHSL, MTS, and GD Constable. The app provides structured practice, timed mock tests, and detailed performance analytics to help students benchmark and improve their ranking.

Platform: Android
Category: Education
Exams: SSC CGL, CHSL, MTS, GD
Language: English

Key Features

Full mock tests with real exam patterns
Subject-wise practice (GK, Math, English, Reasoning)
Timer-based sectional quizzes
Detailed solutions with explanations
Performance dashboard & analytics
Offline practice mode

Built With

Flutter Dart Firebase SQLite REST APIs Android SDK Google Play Console

HappyFlow — Productivity & Wellness Tracker

HappyFlow is a daily productivity and mental wellness application that helps users manage tasks, track emotional states, build positive habits, and maintain a focused mindset. The app integrates structured journaling, habit streaks, and goal-setting tools in a distraction-free interface.

Platform: Android
Category: Productivity
Focus: Habit & Mood Tracking

Key Features

Daily mood & energy check-ins
Habit building with streak tracking
Structured daily journaling
Goal creation & milestone tracking
Progress reports & weekly summaries
Minimal, distraction-free UI

Built With

Flutter Dart Firebase Auth Firestore SQLite Android SDK Google Play Console

Learn Tamil Online — Language Learning Platform

A structured digital learning platform to help non-Tamil speakers learn the Tamil language through progressive lessons, audio pronunciation guides, vocabulary drills, and interactive quizzes. The platform is part of GrindQ's digital learning initiative and is designed to serve the Tamil diaspora globally.

Platform: Android & Web
Category: Education
Audience: Global Tamil Diaspora

Key Features

Structured lesson plan (A1 to B2 levels)
Audio pronunciation with native speakers
Script & reading practice modules
Vocabulary flashcards & spaced repetition
Progress tracking & level tests
Cultural context lessons

Built With

Flutter Dart React / Next.js Firebase Node.js MongoDB AWS S3 Android SDK
